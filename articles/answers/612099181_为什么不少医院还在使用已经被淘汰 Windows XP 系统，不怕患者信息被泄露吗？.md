# 为什么不少医院还在使用已经被淘汰 Windows XP 系统，不怕患者信息被泄露吗？

> 回答：2019-03-02 12:30｜更新：2019-03-02 12:30｜赞同：0｜评论：0
> 原文：https://www.zhihu.com/question/314024154/answer/612099181

---

任何生产环境，企业环境，都是能用就行，能不升级就不升级。

有些10~20年的老交换机，灰尘可以以厘米来计算，只要不坏，还是继续服役。

升级是要考虑投入产出比的，投入人力、金钱等资源换来不带来明显收益的项目，这样的决策，正常的领导都不会做。
