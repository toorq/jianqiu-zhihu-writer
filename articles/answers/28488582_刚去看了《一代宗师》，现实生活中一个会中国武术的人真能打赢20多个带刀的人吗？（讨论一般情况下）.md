# 刚去看了《一代宗师》，现实生活中一个会中国武术的人真能打赢20多个带刀的人吗？（讨论一般情况下）

我是练传统武术的，我也来说两句吧。

要问能不能一个打很多带刀的人，不是一个会的人能不能打倒多少人，和带什么武器的问题，而是会的人的水平到什么水平的问题。

这就好像考试，不是每个学数学的人都能解开一个高难度的题。一打多，还是多个持械的，本来就是个高难度问题。

不过，要达到这种1 vs N 能够赢，拳术要练到一定境界，有几个条件必须具备，这几个条件就好像是水线，没达到，就无法做到1 vs N。自问可以达到了，一打多就有赢的可能：

1）必须能够空手接刀，接棍。至少要不怕刀，不怕棍。面对武器，谁怕谁死。要做到人家，一刀下来，不知道怎么地，刀已经在你一只手里，而已经另外一只手已经放到他的要害上了。看起来难，实际上每个正经的门派都有对付刀，剑等兵器的法门，只是没有电影里什么空手接白刃那么夸张就是了。这点最难达到。

2）功力要足够好，好到什么程度呢，要做到一下撂倒一个，打一个人绝不打第二下，他已经起不来了。这样在搏斗的时候不会出现人越打越多，人打不少，因为毕竟自己是一个人，体力不够用的。做到这点，即使对20人。打20点体力上也是很easy的。而且对敌人震撼性也很大。这点难度低一些。

3）懂的位置，走位好，不被包围。其实一个打20个甚至更多的人，同时面对的就1~2个。要走位，不要被围住，要被围住四面八方都是刀，神仙都没办法。这样虽然对手人多，同时面对的敌人也就1~2个，至少在1秒的时间里是这样。学拳的人都知道，对一个人基本1秒内就能解决战斗了，没有武侠片那么花销。这点难度一般。

4）体力要好，至少打那么几十个人不能喘气不能累。这点最容易达到。

回到刚才说的，不管什么功夫，什么拳派，只有练不好的人，没有不好的拳，因为经过几百年岁月沉淀的拳派经过多少打斗沉淀下来的功夫是现代人很难想象的。

就拿以上四点来说，练传统武术的就很少能够做到。能不能做到，关键还是看人，看功夫练的怎么样。

对于传统武术，大家最大的一个误区，就是好像只要练了一步绝招就可以在很短时间内独步武林天下无敌，做不到，那就是功夫有问题。问问大家自己，学数学学了多少年，学英语、语文、美术、历史、地理这些学科都学了多少年？

武术也是一门科学，需要时间去沉淀，需要汗水去浇灌。你要是问学传统武术的能不能打，我会告诉你能。你问我能不能打赢拳王，能不能打赢10个带刀的，我会告诉你不能。是我不能，而不是中国传统武术不能。

谢谢。

PS：这样持械的恶斗只要不是有组织的，往往要是一招制敌，打趴一个，接下去几个往往就懵了。
