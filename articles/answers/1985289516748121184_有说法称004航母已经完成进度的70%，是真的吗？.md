作为福建人来猜一下，

下一艘航母，应该是福建舰的姐妹舰，

台湾舰。

* * *

看来很多人不了解，我这里补充一下。

1）为什么我国航母从16开始，因为1到15号给了北洋水师：七镇（镇东 / 镇西 / 镇南 / 镇北 / 镇中 / 镇边 / 镇海），八远（定远 / 镇远 / 济远 / 致远 / 靖远 / 经远 / 来远 / 平远）；

2）第一艘航母叫辽宁舰，可能是为了纪念甲午海战；

2）第二艘航母叫山东舰，可能是为了纪念威海卫海战；

4）第三艘航母叫福建舰，可能是为了纪念中海马江海战，同时北洋水师有很多福州人，也和福州马尾造船厂有关系。

5）第四艘航母为什么我觉得很有可能叫台湾舰。第一台湾和福建隔海相望，第二历史上台湾有台江海战，**澎湖海战……等。**
