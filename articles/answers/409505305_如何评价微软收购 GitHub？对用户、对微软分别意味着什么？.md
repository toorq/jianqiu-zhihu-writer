# 如何评价微软收购 GitHub？对用户、对微软分别意味着什么？

> 回答：2018-06-04 23:15｜更新：2018-06-04 23:15｜赞同：0｜评论：0
> 原文：https://www.zhihu.com/question/279538141/answer/409505305

---

对微软来说肯定是利好，微软的没落是在失去MSDN开始的。

对于微软这个级别的公司，打的是整个生态，开发人员是生态中的重要一环。桌面时代微软的成功，MSDN非常重要。苹果在移动时代的成功，IOS开发生态非常重要。Google、Facebook为什么要主导那么多开源项目，Golang、React……都是为了自己的开发生态。

如果说技术是一个战争，那么谁的生态能吸引更多的开发人员，就好像谁打仗人更多。

Github对微软来说随便做做，都大有好处。

1）大数据扒拉扒拉，知道哪些程序员牛逼，HR部门有活干了。

2）新项目扒拉扒拉，看看有什么有前途的开源项目，直接出钱资助，或者买团队，总之要引导技术方向。

3）老项目social下，看看能不能对微软生态更友好点。
